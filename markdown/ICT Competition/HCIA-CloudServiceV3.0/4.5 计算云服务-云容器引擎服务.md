# 云容器引擎

## 是什么

Cloud Container Engine CCE 提供高度可扩展的、高性能的企业级Kubernetes集群，支持运行Docker容器。借助云容器引擎，用户可在华为云上轻松部署、管理和扩展容器化应用程序
